# Logo System & Visual Assets

## 🖼 Logo Variants
- **Full Logo (`logo-full.svg`):** Symbol + Full Name. Use for Desktop Header, Footer, and About pages.
- **Compact Logo (`logo-horizontal.svg`):** Symbol + Short Name ("Checa de Galicia"). Use for Tablet and medium layouts.
- **Icon Mark (`logo-icon.svg`):** Symbol only. Use for Mobile Header (375px - 768px), Favicons, and Mobile Menu.
- **Inverted Logos:** Use `logo-white.svg` on dark backgrounds (if any) and `logo-black.svg` for high-contrast needs.

## 📏 Sizing & Clear Space
- **Mobile Header Height:** 36px (32px–40px range).
- **Desktop Header Height:** 44px (Standard).
- **Clear Space Rule:** Minimum space around the logo must equal the height of the inner symbol. 
- **Exclusion Zone:** No motifs, buttons, or text within 100px (Desktop) or 48px (Mobile) of the logo.

## 🎨 Logo Color Rules
- **Approved Colors:** Navy (#001F3F), Coral (#E5534B), Sky (#5FA8D3).
- **Prohibitions:** No gradients, no drop shadows, no transparency, no color modifications.

## 📁 Favicons & App Icons
- `favicon-32.png` / `favicon-16.png`
- `apple-touch-icon.png` (57px/180px)

## 🅰️ Font Implementation (Zero-Budget Lock)
- **Source:** Google Fonts (Free).
- **Method:** Standard HTML <link> in `index.html`.
- **Families:** - Montserrat:700 (Headings)
  - Open Sans:400,600 (Body)
- **CSS Rule:** Define these as CSS variables in your main Tailwind/CSS file for consistency.
