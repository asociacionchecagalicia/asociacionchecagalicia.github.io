## Czech Cultural Association of Galicia

Official website for the **Asociación Cultural Checa de Galicia**, based in Spain. This project aims to create a minimalist, elegant, and community-focused digital space to promote Czech culture and heritage in the Galician region.

## 🌟 Project Vision
A clean, high-performance platform designed to facilitate cultural exchange, event organization, and community building. The site is built with a "Privacy-First" and "Performance-First" philosophy.

## 🛠 Tech Stack
- **Framework:** Vue 3 (Composition API)
- **Language:** JavaScript (ES6+)
- **Styling:** Tailwind CSS
- **Deployment:** GitHub Pages (Automated via GitHub Actions)
- **i18n:** Full support for Spanish (ES) and Czech (CZ)

## 📂 Architecture Overview
- **Modular Components:** Feature-based structure in `src/components/modules/`.
- **Global Layouts:** Reusable structural blocks in `src/components/layout/`.
- **Design Driven:** Developed strictly following Figma high-fidelity references.
